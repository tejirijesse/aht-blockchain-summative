# Deliverable Checklist

## Included in repository

- Source code in `src/` and `include/`
- Build file in `Makefile`
- Main project overview in `README.md`
- Technical report in `docs/TECHNICAL_REPORT.md`
- Design diagrams in `docs/SYSTEM_DESIGN_DIAGRAM.md`
- Demo plan in `docs/DEMO_VIDEO_PLAN.md`
- Command mapping in `docs/COMMAND_MAPPING.md`

## Before submission

- Ensure GitHub repo is public
- Confirm latest push is on `main`
- Record demo video using the plan in `docs/DEMO_VIDEO_PLAN.md`
- Export Mermaid diagrams to image or PDF if your instructor prefers visual files
- If needed, convert the technical report into PDF or Word format

## Recommended submission bundle

1. GitHub repository link
2. PDF of technical report
3. PDF or PNG of system design diagrams
4. Demo video link or uploaded file
5. Short submission note summarizing the project
